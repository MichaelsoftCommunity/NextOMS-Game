# 架空世界地图模拟器

这是一个基于网页的架空世界地图模拟器，允许用户创建自定义国家、设计地图并模拟国家发展。

## 功能特点

- 创建自定义国家，设置名称、颜色、政府类型等属性
- 编辑地图，为国家分配领土
- 模拟国家发展，包括人口增长、资源变化和技术进步
- 交互式地图界面，支持拖动和缩放
- 实时显示国家和世界统计数据

## 使用方法

1. 点击"创建国家"按钮，填写国家信息
2. 点击"编辑地图"按钮进入编辑模式
3. 在地图上点击为选中的国家添加领土
4. 点击"开始模拟"按钮开始世界模拟
5. 使用"+"和"-"按钮或鼠标滚轮缩放地图
6. 点击地图上的国家领土查看详细信息

## 技术实现

- 纯原生 HTML5、CSS3 和 JavaScript 实现
- 使用 Canvas API 绘制交互式地图
- 模拟算法包括人口增长、资源管理和国家互动
